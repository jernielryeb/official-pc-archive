# official-pc-archive

本仓库用于归档和发布多个独立 HTML 页面，旨在长期保存 Web 内容的静态快照。

## 项目简介

`official-pc-archive` 是一个以 HTML 页面为单位的归档仓库。它不针对任何单个域名或具体网站，而是作为一系列独立页面的存储与发布集合。每个页面均以完整的 HTML 格式保存，保留原始的结构与样式，便于离线查看或分发。

## 目录说明

仓库根目录下按页面名称或时间组织目录结构，典型布局如下：

```
official-pc-archive/
├── page-001/
│   ├── index.html
│   └── assets/          # 页面相关资源（图片、样式、脚本等）
├── page-002/
│   └── index.html
└── README.md
```

每个页面独立为一个子目录，目录名通常反映页面内容或归档批次。建议为每个页面附带必要的资源文件，以保持页面完整性。

## 页面归档说明

- 归档内容：仅包含 HTML 文件及其所需静态资源（CSS、JS、图片等），不包含动态数据或后端依赖。
- 页面来源：来自互联网公开内容，或由贡献者制作/整理。
- 文件格式：所有页面使用标准 HTML5 格式，编码统一为 UTF-8。
- 归档原则：保持页面原始面貌，不修改内容、不添加额外链接或追踪代码。

## 维护说明

- 本仓库由维护者不定期更新，不保证归档频率。
- 欢迎提交 Issue 或 Pull Request 建议新增页面，但需确保内容合法合规，且不侵犯他人权益。
- 如发现页面无法正常打开或资源缺失，可提交 Issue 说明具体页面路径。
- 仓库不会主动删除已归档页面，除非因法律或政策要求。
